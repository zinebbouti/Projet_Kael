ğŸ¯ Problem Statement & Context

SAVEWOOD is revolutionizing the qualification of understory trees using mobile LiDAR. However, the preliminary identification of strategic areas of interest at the plot scale remains a major challenge when performed from the ground.

The CHP project provides an innovative solution by delivering a synthetic and precise aerial overview to intelligently guide field campaigns and drastically optimize measurement time, while improving the quality of the collected data.

ğŸ”¬ Detailed methodology

Mission planning

Automatic and intelligent generation of waypoints from a user-defined polygon. Optimized computation accounting for flight altitude, frontal and lateral overlap, and sensor optical parameters to guarantee full coverage.

3D reconstruction

Advanced photogrammetric processing with WebODM to produce a high-resolution georeferenced orthophoto and a dense 3D point cloud enabling accurate canopy analysis.

lidR analysis

Comprehensive extraction of forest indicators: DTM (Digital Terrain Model), CHM (Canopy Height Model), automatic individual tree detection, and accurate height estimation using advanced segmentation algorithms.
